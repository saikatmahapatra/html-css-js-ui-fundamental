Event Bubbling & Event Capturing? How to prevent? (Bubble Up, Trickle Down)

Event bubbling and capturing are two ways of event propagation in the HTML DOM API, when an event occurs in an element inside another element, and both elements have registered a handle for that event. The event propagation mode determines in which order the elements receive the event.

With bubbling, the event is first captured and handled by the innermost element and then propagated to outer elements. ****=> inner->outer in DOM tree

IE < 9 uses only event bubbling, whereas IE9+ and all major browsers support both.

To prevent we have to write e.stopPropagation();
Also e.stopImmediatePropagation();
